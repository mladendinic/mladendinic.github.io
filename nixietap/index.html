Nixie Tap is a minimalistic Nixie clock, designed to showcase the elegance of Soviet IN-12B Nixie tubes.

Each enclosure is carved out of a solid block of walnut, hand polished and sealed with oil. This makes each unit unique.

It is available as a kit and as a finished unit here. If you want to build everything by yourself, please go to Downloads page.

Features:

  • direct drive, no PWM flickering or audible noise
  • USB powered, meets USB power draw specification
  • NTP time synchronization
  • touch sensor based interaction
  • open source design
